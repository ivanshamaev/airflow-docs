# Прочее (Others)

Раздел с материалами по инструментам и темам, полезным при работе с Airflow: менеджер пакетов uv, линтер Ruff, тестовый фреймворк pytest, типчекер ty и основы Pydantic.

| Раздел | Описание |
|--------|----------|
| [Менеджер пакетов (uv)](package-manager/README.md) | uv — быстрый менеджер пакетов и проектов для Python (установка, проекты, зависимости, туториалы) |
| [Linter (Ruff)](linter/README.md) | Линтер и форматтер Ruff: установка, правила, интеграция в проект, правила для Airflow (AIR) |
| [pytest](pytest/README.md) | Тестовый фреймворк pytest: обзор, запуск, утверждения, фикстуры, маркировка, параметризация, subtests, monkeypatch, doctest |
| [ty](ty/README.md) | Сверхбыстрый типчекер и language server ty (Astral): установка, проверка типов, интеграция с редакторами, конфигурация, правила, система типов, CLI |
| [Pydantic](Pydantic/README.md) | Валидация данных, модели, типы, работа с файлами, API и базами данных |
