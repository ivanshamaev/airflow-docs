# Получение помощи

## Справка по командам

Флаг `--help` показывает справку по команде, например для `uv`:

```bash
$ uv --help
```

Чтобы увидеть справку по конкретной команде, например по `uv init`:

```bash
$ uv init --help
```

С флагом `--help` uv выводит краткую справку. Чтобы увидеть расширенную справку по команде, используйте `uv help`:

```bash
$ uv help
```

Расширенная справка по конкретной команде, например по `uv init`:

```bash
$ uv help init
```

При расширенной справке uv пытается использовать `less` или `more` для постраничного вывода. Чтобы выйти из пейджера, нажмите `q`.

## Подробный вывод

Флаг `-v` включает подробный вывод команды, например для `uv sync`:

```bash
$ uv sync -v
```

Флаг `-v` можно повторять для увеличения уровня детализации:

```bash
$ uv sync -vv
```

Подробный вывод часто содержит дополнительную информацию о том, почему uv ведёт себя тем или иным образом.

## Проверка версии

При обращении за помощью важно знать версию uv — иногда проблема уже исправлена в более новой версии.

Проверить установленную версию:

```bash
$ uv self version
```

Также допустимы варианты:

```bash
$ uv --version      # То же, что `uv self version`
$ uv -V             # Без коммита сборки и даты
```

**Примечание:** До uv 0.7.0 использовалась команда `uv version` вместо `uv self version`.
