# Менеджер пакетов (uv)

Раздел с переводом документации [uv](https://docs.astral.sh/uv/) — быстрого менеджера пакетов и проектов для Python (Astral, Rust).

## Содержание раздела

- **[Введение в uv](uv-introduction.md)** — установка, проекты, скрипты, инструменты, версии Python, интерфейс pip.
- **[Установка uv](installation.md)** — автономный установщик, PyPI, Homebrew, MacPorts, WinGet, Scoop, Docker, GitHub Releases, Cargo; обновление, автодополнение в оболочке, удаление.
- **[Возможности uv](uv-features.md)** — версии Python, скрипты, проекты, инструменты, интерфейс pip, служебные команды.
- **[Получение помощи](uv-help.md)** — справка по командам, подробный вывод, проверка версии, устранение неполадок, GitHub, Discord.
- **[Туториал: Python UV (DataCamp)](python-uv-tutorial.md)** — сравнение с pip, Conda, Poetry; начало работы, управление версиями Python, инструменты (uvx), lockfile, продвинутое управление зависимостями, миграция с pip/virtualenv, FAQ.
