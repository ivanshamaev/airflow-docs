# Pendulum

Раздел с переводом документации библиотеки [Pendulum](https://pendulum.eustace.io/) — удобной работы с датой и временем в Python (замена и расширение стандартного `datetime`).

## Содержание раздела

- **[Введение и документация](introduction.md)** — установка, введение, создание экземпляров (`datetime()`, `now()`, `local()`, `from_format()`, `from_timestamp()`), разбор строк (RFC 3339, ISO 8601), локализация, атрибуты и свойства, fluent-помощники, форматирование в строку, сравнение, сложение и вычитание, разница и `diff_for_humans()`, модификаторы, часовые пояса, Duration, Interval, тестирование времени, ограничения и обходы (sqlite3, MySQL, Django).
