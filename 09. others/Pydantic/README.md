# Pydantic

Русский перевод документации [Pydantic](https://docs.pydantic.dev/) — библиотеки валидации данных для Python.

## Содержание раздела

1. **[Добро пожаловать в Pydantic](welcome-to-pydantic.md)** — введение, зачем использовать Pydantic, мониторинг с Logfire, примеры.

2. **[Модели (Models)](Models.md)** — определение схем через модели, базовое использование, преобразование данных, вложенные модели, валидация, дженерики, RootModel, замороженные модели и др.

3. **[Поля (Fields)](fields.md)** — настройка полей: значения по умолчанию, алиасы, ограничения, строгие поля, дискриминаторы, устаревшие поля, вычисляемые поля (`computed_field`).

4. **[Типы (Types)](types.md)** — кастомные типы, аннотированный паттерн, именованные алиасы, рекурсивные типы, `__get_pydantic_core_schema__`, дженерик-классы.

5. **[Валидация данных из файлов](validating-file-data.md)** — примеры валидации данных из JSON, JSONL, CSV, TOML, YAML, XML и INI.

6. **[Веб- и API-запросы](requests.md)** — валидация ответов HTTP с помощью моделей Pydantic (пример с httpx и JSONPlaceholder).

7. **[Очереди (Queues)](queues.md)** — валидация и сериализация данных при работе с очередями: Redis, RabbitMQ, ARQ.

8. **[Базы данных (Databases)](databases.md)** — использование Pydantic с ORM на примере SQLAlchemy (схемы, алиасы, дублирование кода).
