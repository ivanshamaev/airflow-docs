# deepwiki-airflow-ru

Русский перевод документации **Apache Airflow®** в формате Markdown, структура соответствует [deepwiki.com/apache/airflow](https://deepwiki.com/apache/airflow). Для личного использования и размещения на GitHub.

## Структура

```
apache/
  airflow/
    README.md           # Оглавление документации
    index.md            # Что такое Airflow?
    quick-start.md      # Быстрый старт
    core-concepts/
      README.md         # Основные концепции — оглавление
      dags.md           # DAG
      tasks.md          # Задачи (Tasks)
```

## Использование

- Читать в любом Markdown-редакторе или на GitHub (удобна навигация по ссылкам).
- Исходный текст взят из [официальной документации Apache Airflow](https://airflow.apache.org/docs/); перевод неофициальный.
- Контент DeepWiki (deepwiki.com/apache/airflow) подгружается через JavaScript, поэтому тексты брались с airflow.apache.org; последовательность разделов и разделы «Основные концепции» приведены в соответствие с типичной структурой документации Airflow.
